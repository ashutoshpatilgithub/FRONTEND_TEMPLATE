# 📚 Bookworm — Library Management System

A full-stack MERN Library Management System with authentication, book management, and borrowing features.

## Project Structure

```
bookworm/
├── frontend/       # React + Vite + Redux + Tailwind
├── backend/        # Node.js + Express + MongoDB
└── package.json    # Root scripts
```

## Tech Stack

| Frontend        | Backend         |
|-----------------|-----------------|
| React 18        | Node.js         |
| Redux Toolkit   | Express         |
| Tailwind CSS    | MongoDB         |
| React Router    | JWT Auth        |
| Vite            | Cloudinary      |

## Setup

```bash
# Install all dependencies
npm run install-all

# Start both servers
npm run dev
```

## Scripts

| Command           | Description                      |
|-------------------|----------------------------------|
| `npm run dev`     | Start both frontend & backend    |
| `npm run frontend`| Start only the frontend          |
| `npm run backend` | Start only the backend           |
| `npm run studio`  | Open Prisma Studio (data viewer) |
